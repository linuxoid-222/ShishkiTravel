    # Италия: визы, въезд, ключевые правила для туристов

    > Этот документ — справка для туристического бота. Правила могут меняться. Всегда проверяйте актуальность по официальным источникам ниже.

    ## Визы и въезд (для граждан РФ)

    - Для граждан РФ для туризма **нужна шенгенская виза (C)**.
- Официальный справочник по визам: Il Visto per l'Italia.

    ## Ключевые условия въезда / документы

    - Типовой пакет: паспорт, анкета, страховка, подтверждения проживания/маршрута/финансов.
- Проверьте требования и список документов на официальном визовом портале.

    ## Важные правила и ограничения (кратко)

    - Запрещены наркотики; соблюдайте местные правила (особенно в исторических центрах и на транспорте).

    ## Источники (официальные)
