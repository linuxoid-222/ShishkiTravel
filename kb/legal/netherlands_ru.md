    # Нидерланды: визы, въезд, ключевые правила для туристов

    > Этот документ — справка для туристического бота. Правила могут меняться. Всегда проверяйте актуальность по официальным источникам ниже.

    ## Визы и въезд (для граждан РФ)

    - Для граждан РФ для туризма **обычно нужна шенгенская виза (C)**.
- Официальный справочник по визам Нидерландов: NetherlandsWorldwide.

    ## Ключевые условия въезда / документы

    - Типовой пакет: паспорт, страховка, подтверждение проживания/маршрута/финансов.
- Проверяйте актуальные требования и куда подавать документы на NetherlandsWorldwide.

    ## Важные правила и ограничения (кратко)

    - Запрещены наркотики и их оборот; соблюдайте правила общественного порядка.

    ## Источники (официальные)

    - https://www.netherlandsworldwide.nl/visa-the-netherlands/schengen-visa/apply-russian-federation
