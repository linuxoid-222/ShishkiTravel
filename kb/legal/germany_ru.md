    # Германия: визы, въезд, ключевые правила для туристов

    > Этот документ — справка для туристического бота. Правила могут меняться. Всегда проверяйте актуальность по официальным источникам ниже.

    ## Визы и въезд (для граждан РФ)

    - Для граждан РФ для туризма **обычно нужна шенгенская виза (C)**.
- Официальная информация по въезду/визам публикуется немецким МИД (Auswärtiges Amt).

    ## Ключевые условия въезда / документы

    - Паспорт, страховка, подтверждение цели поездки/проживания/финансов — типовой пакет.
- Проверяйте ограничения и актуальные правила въезда на странице МИД Германии.

    ## Важные правила и ограничения (кратко)

    - Запрещены наркотики; за нарушения — уголовная/административная ответственность.
- В общественных местах действуют местные правила (транспорт, штрафы, порядок).

    ## Источники (официальные)
