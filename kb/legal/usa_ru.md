    # США: визы, въезд, ключевые правила для туристов

    > Этот документ — справка для туристического бота. Правила могут меняться. Всегда проверяйте актуальность по официальным источникам ниже.

    ## Визы и въезд (для граждан РФ)

    - Для граждан РФ для туризма **обычно нужна виза B-2** (Visitor Visa).
- Официальная информация и анкеты — на сайте Госдепартамента США.

    ## Ключевые условия въезда / документы

    - Подача: заполнение DS-160, оплата сбора, запись на собеседование (процедуры могут отличаться по стране подачи).
- Рекомендуется проверять актуальные требования на travel.state.gov.

    ## Важные правила и ограничения (кратко)

    - Соблюдайте местные законы (они различаются по штатам).
- Не нарушайте сроки пребывания/условия визы.

    ## Источники (официальные)

    - https://travel.state.gov/content/travel/en/us-visas/tourism-visit/visitor.html
