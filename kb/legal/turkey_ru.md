    # Турция: визы, въезд, ключевые правила для туристов

    > Этот документ — справка для туристического бота. Правила могут меняться. Всегда проверяйте актуальность по официальным источникам ниже.

    ## Визы и въезд (для граждан РФ)

    - Для граждан РФ для кратких туристических поездок действует **безвизовый режим до 60 дней** (с ограничениями по сроку пребывания и требованиям к паспорту).

    ## Ключевые условия въезда / документы

    - Важно: паспорт должен быть действительным достаточный срок на дату въезда; официальные разъяснения для россиян публикуют консульские источники РФ в Турции.

    ## Важные правила и ограничения (кратко)

    - Соблюдайте местные законы; запрещены наркотики.
- В отдельных местах могут быть ограничения на фото/видео (военные объекты, госучреждения).
