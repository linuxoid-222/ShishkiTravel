    # Испания: визы, въезд, ключевые правила для туристов

    > Этот документ — справка для туристического бота. Правила могут меняться. Всегда проверяйте актуальность по официальным источникам ниже.

    ## Визы и въезд (для граждан РФ)

    - Для граждан РФ для туризма **нужна шенгенская виза (C)**.
- Информация по визам публикуется МИД Испании/посольствами и визовыми центрами.

    ## Ключевые условия въезда / документы

    - Типовой пакет: паспорт, анкета, фото, страховка, подтверждения поездки/финансов.
- Актуальные требования смотрите на официальных страницах МИД/посольства.

    ## Важные правила и ограничения (кратко)

    - Запрещены наркотики; соблюдайте общественный порядок и местные правила.

    ## Источники (официальные)
